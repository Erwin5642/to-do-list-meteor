# 📝 Meteor To-Do List App

A simple and responsive **To-Do List application** built with **Meteor** that allows users to create, manage, and organize daily tasks in real time.

This project demonstrates the core features of Meteor such as **reactive data updates, client-server integration, and full-stack JavaScript development**.

---

##  Features

-  Add new tasks
-  Mark tasks as completed
-  Delete tasks
-  Real-time updates across clients
-  Simple and responsive UI
-  Reactive data updates with Meteor

---

##  Tech Stack

- **Framework:** Meteor  
- **Language:** JavaScript  
- **Database:** MongoDB (via Meteor)  
- **Frontend:** HTML, CSS, Blaze / React (depending on implementation)

---

##  Installation

### 1. Install Meteor

Mac / Linux:

```bash
curl https://install.meteor.com/ | sh
```
Windows users can follow the official Meteor installation guide.

### 2. Clone the Repository
```bash
git clone https://github.com/yourusername/meteor-todo-app.git
cd meteor-todo-app
```

### 3. Install Dependencies
```bash
meteor npm install
```

### 4. Run the Application
```bash
meteor run
```
